# Pretty Print — Free Online Code Formatter & Beautifier

**Pretty Print** is a premium, open-source code formatting tool designed for developers who value speed, privacy, and aesthetics. It supports over 25+ programming languages and processes everything **100% client-side**, ensuring your code never leaves your browser.

![Pretty Print Showcase](https://prettyprint.online/og-image.png)

## ✨ Features

- **25+ Languages Supported**: JSON, SQL, HTML, CSS, JavaScript, TypeScript, Python, Java, C++, GraphQL, and more.
- **100% Client-Side**: Your code is processed entirely in your browser using JavaScript and WebAssembly. No server uploads.
- **Premium UX**: Dark-mode first design with glassmorphism effects and smooth transitions.
- **Smart Formatting**: Intelligent indentation, minification, and syntax highlighting.
- **SEO Optimized**: Built-in JSON-LD schema, dynamic sitemaps, and OpenGraph metadata for perfect search engine visibility.
- **PWA Ready**: Installable on mobile and desktop for offline-ready access.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Editor Engine**: [Monaco Editor](https://microsoft.github.io/monaco-editor/)
- **SEO**: Comprehensive JSON-LD & OpenGraph

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/pretty-print.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
